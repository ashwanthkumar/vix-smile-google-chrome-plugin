# IVSmile

## Development
Need to setup the tailwind to run locally while development

```bash
$ npm i
$ npx tailwindcss -i ./css/input.css -o ./ivsmile.css --watch
```

## Build
```bash
$ npx tailwindcss -i ./css/input.css -o ./ivsmile.css
```
